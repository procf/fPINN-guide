# 3D fPDE case

This folder contains the solver and plotter notebooks for the three-dimensional fractional PDE case (Section 3.3 of the paper).

**Note:** Pre-trained model weights for this case will be added prior to publication.

## Files

- `3DfPDE_solver.ipynb` — trains the fPINN model from scratch
- `3DfPDE_plotter.ipynb` — loads saved weights and regenerates paper figures (requires `WEIGHTS_*/` folders, to be added)
